Connection avec VNC¶

Connection avec SSH¶

Voir: https://www.raspberrypi.org/documentation/computers/remote-access.html

  • login: pi

  • password: raspberry

  • 192.168.2.4 - 192.168.1.5

Utiliser -Y pour exporter le serveur graphique
ssh -Y pi@192.168.2.4
ssh pi@192.168.2.4
ssh -L 8889:localhost:8888 pi@192.168.2.4

ssh -L 8889:localhost:8888 pi@169.254.186.223

Setup wifi¶

sudo raspi-config

sudo nano /etc/wpa_supplicant/wpa_supplicant.conf

ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
update_config=1
country=GB

network={
 ssid="sciento_2.4Ghz"
 scan_ssid=1
 psk="password"
 key_mgmt=WPA-PSK
}


network={
 ssid="Le Terrier_5"
 scan_ssid=1
 psk="password"
 key_mgmt=WPA-PSK
}

network={
 ssid="AMERICA"
 scan_ssid=1
 psk="password"
 key_mgmt=WPA-PSK
}
